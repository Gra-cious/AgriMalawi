# AgriMalawi

**Project Code:** AGM-001

AgriMalawi is a mobile-first digital agriculture platform designed to help farmers and aspiring farmers successfully plan, budget, cultivate, and manage Crop Projects.

## Vision

To empower every farmer with practical agricultural knowledge through technology.

## Technology Stack

- Next.js
- TypeScript
- Tailwind CSS
- Django
- PostgreSQL
