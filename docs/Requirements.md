## Functional Requirements

The system shall:

- Allow users to browse Crop Projects.
- Allow users to search for crops.
- Display complete farming guides.
- Display project budgets.
- Display expected yields.
- Display fertilizer schedules.
- Display irrigation schedules.
- Display pest and disease management.
- Display herbicide and pesticide recommendations.
- Allow users to register.
- Allow users to log in.
- Allow administrators to manage content.

## Non-Functional Requirements

- Mobile-first.
- Responsive design.
- Fast loading.
- Secure authentication.
- Scalable architecture.
- Easy navigation.
